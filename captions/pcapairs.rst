**PC Scatter Plot Matrix (Pairs Plot)**
---------------------------------------
A cross-comparison between the top principal components (e.g., PC1 vs PC2, PC1 vs PC3, PC2 vs PC3). 
It provides a multidimensional view of the dataset, useful if group separation is not 
entirely captured by the first two components alone.
